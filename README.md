# Homunculus: A [bad] Discord bot for Eve-Online

## Features
- Price look-up (only Jita for now, region and system support is coming. 
- Quick character look-up. (Things like current corp/alliance, K/D)

## Requirements
- Python 3.5
- `discord.py`
